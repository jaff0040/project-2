# ⌚ Elite Time – Luxury Watch Store

Elite Time is a modern luxury watch website built using HTML5 and CSS3.  
The project showcases premium watch collections with an elegant and responsive-inspired layout design.

## ✨ Features

- Modern luxury watch design
- Animated navigation hover effects
- Dropdown menu
- Hero section with promotional banner
- Trending watch products section
- Hover image effects for products
- Customer testimonial section
- Latest news section
- Delivery and support services
- Professional footer with newsletter subscription

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome Icons

---

## 🎨 Design Style

The website uses a luxury-inspired color palette based on:

- Dark Navy Blue
- Light Blue
- White
- Soft Gray

The UI design focuses on elegance, clean layouts, and smooth hover animations.
